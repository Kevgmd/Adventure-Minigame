### Minijogo de Aventura Por Kevgmd
- Jogo feito em formato de Visual Novel, com o objetivo sendo de encontrar o seu caminho de volta para casa em um mundo desconhecido;
- Múltiplas escolhas de caminhos para seguir;
- Paisagens exuberantes (Feitas com IA);
- Trilha Sonora própria em sua maior parte;
- Inteiramente feito em HTML, CSS e polido com Javascript.
