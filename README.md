### Minijogo de Aventura Por Kevgmd
- Jogo feito em formato de Visual Novel, com o objetivo sendo de encontrar o seu caminho de volta para casa em um mundo desconhecido;

- Múltiplas escolhas de caminhos para seguir, escolhendo de forma sensata;

- Paisagens exuberantes com trilhas sonoras igualmente incríveis;
  
- Inteiramente feito em HTML, CSS e um toque de Javascript.
